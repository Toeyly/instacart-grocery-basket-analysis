# Instacart Grocery Basket Analysis (Python)

## Project Overview
Instacart is an online grocery platform looking to better understand customer purchasing behavior and sales patterns.  
This project performs exploratory analysis and segmentation to support targeted marketing strategies.

## Stakeholder Questions
- What are the busiest days of the week and hours of the day for orders?
- Are there specific times of day when customers spend more?
- How can we group products into simpler price ranges?
- Which departments are most frequently ordered?
- How do ordering behaviors differ by customer loyalty, region, age, and family status?

## Tools
- Python (Jupyter Notebook)
- pandas, NumPy, matplotlib
- (Optional: scipy / seaborn depending on your analysis)

## Key Skills Demonstrated
- Data cleaning: missing values, duplicates, mixed types
- Subsetting and filtering
- Merging multiple datasets with merge flags
- Feature engineering (derived columns / flags)
- Grouping & aggregation (groupby)
- Visualization (histogram, bar, line, scatter)
